# WordQuest - Game Edukasi Bahasa Indonesia

**WordQuest** adalah permainan edukatif berbasis web yang digunakan untuk mempelajari kosakata, tata bahasa, dan pengetahuan kebahasaan Bahasa Indonesia secara interaktif. Pemain dapat memilih tiga mode permainan yaitu Susun Kata (menyusun huruf acak menjadi kata yang benar), Cocokkan Kata (memasangkan kata dengan artinya), dan Kuis Kilat (menjawab pertanyaan pilihan ganda dengan batasan waktu).


## Fitur Utama

| Fitur | Keterangan |
|-------|------------|
| **3 Mode Permainan** | Susun Kata, Cocokkan Kata, dan Kuis Kilat |
| **10 Level Kesulitan** | Pemula (Level 1) hingga Master (Level 10) |
| **Sistem Nyawa** | 3 kesempatan menjawab sebelum Game Over |
| **Sistem Bintang** | Penilaian berdasarkan persentase jawaban benar |
| **Skor Tertinggi** | Tersimpan secara otomatis di perangkat |
| **Fitur Suara** | Background music dan efek suara interaktif |
| **Tutorial Interaktif** | Panduan bermain setiap mode game |


## Cara Bermain

1. **Pilih Mode Game** pada halaman utama:
   - **Susun Kata** - Susun huruf acak menjadi kata yang benar
   - **Cocokkan Kata** - Pasangkan kata dengan artinya
   - **Kuis Kilat** - Jawab pertanyaan dengan batasan waktu

2. **Pilih Level** (1-10) sesuai kemampuan Anda:
   - Level 1-3: Pemula (Mudah)
   - Level 4-6: Menengah (Sedang)
   - Level 7-10: Mahir (Sulit)

3. **Mulai Bermain** dan kumpulkan poin sebanyak-banyaknya!


## Tujuan Pembelajaran

Setelah bermain WordQuest, pemain diharapkan dapat:
- Meningkatkan pemahaman kosakata Bahasa Indonesia
- Memahami sinonim, antonim, dan makna kata
- Mengenal tata bahasa dan EYD (Ejaan Yang Disempurnakan)
- Mengetahui peribahasa dan ungkapan dalam Bahasa Indonesia
- Meningkatkan kemampuan berpikir kritis melalui kuis
